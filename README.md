# Advanced SQL Sales Analysis (PostgreSQL)

## Overview
This project demonstrates advanced SQL data analysis techniques using PostgreSQL on a real-world sales dataset.

## Key Analysis Performed
- Top 10 Customers by Revenue
- Monthly Revenue Trend
- Profit by Region
- Best Selling Categories
- Average Order Value
- Top Profitable Products
- Customer Ranking using Window Functions
- Yearly Revenue Analysis
- Discount Analysis

## Skills Demonstrated
- GROUP BY & Aggregations
- Date Functions (DATE_TRUNC, DATE_PART)
- Window Functions (RANK)
- Business KPI Extraction
- Performance-Oriented Queries

## Tools Used
- PostgreSQL
- pgAdmin

## Dataset
Superstore Sales Dataset (9,994 records)

## Business Insights
The analysis identifies high-value customers, top-performing regions, revenue trends over time, and profitability drivers across segments.
